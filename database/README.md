# database

## Project setup

```bash
yarn install
```

## Upgrade migrations production

```bash
yarn up
```

## Upgrade migrations development

```bash
yarn dev_up
```

## Downgrade migrations production

```bash
yarn down
```

## Downgrade migrations development

```bash
yarn dev_down
```

## Reset database

```bash
yarn dev_reset
```

Runs all down migrations and after this all up migrations.
